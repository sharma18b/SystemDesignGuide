# CDN Network - Scale and Constraints

## Global Scale
- **Edge Locations**: 200+ POPs worldwide
- **Total Requests**: 10 trillion requests/month
- **Peak Traffic**: 100Tbps aggregate bandwidth
- **Cached Content**: 10PB across all edges
- **Concurrent Users**: 2 billion users

## Per-Edge Capacity
- **Servers per Edge**: 100-1000 servers
- **Bandwidth**: 100Gbps-1Tbps per edge
- **Storage**: 50TB-500TB per edge
- **Requests**: 1M-10M requests/sec per edge

## Performance Targets
- **Cache Hit Rate**: >90%
- **Latency**: <50ms to edge
- **TTFB**: <100ms
- **Availability**: 99.99%

## Cost
- **Infrastructure**: $50M/month
- **Bandwidth**: $20M/month
- **Total**: $70M/month
- **Cost per Request**: $0.0000007
